Die Angaben sind anonym. Dafür benutzen wir nicht Ihren Namen, sondern einen „Code“.

Geben Sie als Code bitte:

  • die ersten beiden Buchstaben des Vornamens Ihrer Mutter an,
  • den Tag Ihres Geburtsdatums
  • und die ersten beiden Buchstaben Ihres Vornamens.

    • Beispiel: AN08PE für Anna (Vorname Ihrer Mutter), 08. Januar 1999 (Ihr eigenes Geburtsdatum), Peter (Ihr eigener Name)

    Bitte den eigenen Code nach obigem Muster in das Textfeld eintragen.
